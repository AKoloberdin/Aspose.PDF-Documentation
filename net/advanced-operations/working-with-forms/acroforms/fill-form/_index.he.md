## מילוי שדה טופס במסמך PDF

כדי למלא שדה טופס, קבל את השדה מאוסף הטפסים של אובייקט המסמך. לאחר מכן הגדר את ערך השדה באמצעות תכונת ה-Value של השדה.

דוגמה זו בוחרת TextBoxField ומגדירה את ערכו באמצעות תכונת ה-Value.

```csharp
// לדוגמאות מלאות וקבצי נתונים, אנא עבור אל https://github.com/aspose-pdf/Aspose.PDF-for-.NET
// הנתיב לספריית המסמכים.
string dataDir = RunExamples.GetDataDir_AsposePdf_Forms();

// פתח מסמך
Document pdfDocument = new Document(dataDir + "FillFormField.pdf");

// קבל שדה
TextBoxField textBoxField = pdfDocument.Form["textbox1"] as TextBoxField;

// שנה ערך שדה
textBoxField.Value = "ערך למילוי בשדה";
dataDir = dataDir + "FillFormField_out.pdf";
// שמור את המסמך המעודכן
pdfDocument.Save(dataDir);
```
